**Инструкция к использованию:**
1. Необходимо установить с github WAF `modsecurity`.

Ссылка на гит:
```
https://github.com/coreruleset/modsecurity-crs-docker/tree/main 
```

2. Пушим себе на гит этот репозиторий. Они должны лежать в одной общей папки, но сам сайт и WAF лежать в разных. Пример:
```
user in /opt 
❯ ls
app    waf
```
Где `app` и `waf` папки с сайтом и WAF соответственно.

3. Устанавливаем docker compose если не скачивали его до этого.
4. Собираем сеть докеров. Сначала запускаем контейнер с WAF, затем с сайтом.
5. После этого заходим в браузере по локальной ссылке вашего устройства и можем тестировать сайт с WAF.

**UPD:**
`docker-compose.yaml` для WAF, `docker-compose.yml` для сайта.
